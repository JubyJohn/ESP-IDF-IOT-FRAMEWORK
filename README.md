# ESP-IDF-IOT-FRAMEWORK
TO FAMILARIZE ESP - IDF FRAMEWORK

##### INSTALLATION:
* REFER: https://docs.espressif.com/projects/esp-idf/en/stable/esp32/get-started/index.html
* MANUAL INSTALLATION -> WINDOWS INSTALL DOWNLOAD
* VS CODE -> EXTENSION -> Search ESP-IDF -> INSTALL
* SETUP ENVIRONMENT -> CUSTOM INSTALLATION

##### CREATE NEW PROJECT:
* ESP-IDF POWERSHELL -> (Type) code-> VS Code OPENS
* Left side Expressif icon -> Advanced -> New project wizard -> select folder
* new project -> ESP-IDF template -> Create project using template sample project
* fill everything accordingly
